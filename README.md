# CookPoint - Online E-Commerce Food Ordering System

## Overview

CookPoint is a dynamic **e-commerce web-based food ordering platform** that enables customers to browse food items, place orders, and track their purchases through an interactive and user-friendly interface. The system includes separate modules for users and administrators, ensuring efficient management of products, orders, customers, and reports.

## Features

* User Registration and Login
* Food Menu Management
* Dynamic Product Display
* Shopping Cart and Order Placement
* Secure Database Integration
* Separate User and Admin Panels
* Order and Customer Management
* Report Generation System
* Responsive and Interactive Interface
* Multiple Database Connectivity (6 Databases)

## Technologies Used

* HTML5
* CSS3
* Java
* MySQL
* XAMPP Server

## System Modules

### User Module

* Account Management
* Food Browsing
* Cart Management
* Order Placement
* Order History

### Admin Module

* Dashboard Management
* Product Management
* Customer Management
* Order Monitoring
* Report Generation

## Project Objective

The objective of CookPoint is to provide a complete e-commerce online food ordering solution that simplifies food purchasing for customers while enabling administrators to efficiently manage business operations through a centralized dashboard.
